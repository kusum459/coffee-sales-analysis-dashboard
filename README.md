Coffee Sales Analysis Dashboard
An interactive Excel dashboard built using advanced Excel techniques to analyze coffee shop sales performance across categories, branches, cities, and time periods, helping stakeholders track revenue, profit, and transaction trends at a glance.

Overview
This dashboard consolidates coffee sales data into a single, interactive Excel view for quick, data-driven decision-making. It highlights key performance indicators alongside visual breakdowns by product category, branch, city, and time (weekly/monthly/yearly), with dynamic filtering for deeper analysis, all built without any external BI tools.

Key Metrics (KPIs)
Transaction ID: 9,971
Quantity Sold: 14,117
Revenue: 46,374.25k
Profit: 16,230.99k

Dashboard Features
Yearly Sales Breakdown: Pie chart comparing sales across 2021, 2022, and 2023.
Branch by Sales: Horizontal bar chart ranking branches by total sales.
Category by Revenue: Bar chart comparing revenue across product categories (Coffee, Cold Drinks, Desserts, Sandwiches, Snacks, Specialty Drinks, Tea).
City by Revenue: Donut chart showing revenue distribution across cities (Austin, Chicago, Los Angeles, New York, and more).
Weekly Transactions: Line chart tracking transaction volume across the days of the week.
Revenue-wise Profit: Bar chart comparing profit generated per city.
Month by Sales: Area chart showing sales trends across all 12 months.
Filter Panel: Interactive filters for Product Category, Branch, and Date (with month/year drill-down) to slice the data dynamically.
